# Create Twilight App: Create a React app using Twilight Cli

致力于为前端开发者快速构建一个现代react项目

## 集成工具

- 前端框架: react
- 打包工具: vite
- 支持Typescript
- react-route路由
- 第三方UI库: antd
- 支持tailwindcss
- 状态管理工具: zustand
- clsx, prettier, immer, ...

## 实用功能

- 深色模式
- 中/英文切换
- 移动端网页

## 如何使用

```shell
npm i -g create-twilight-app
```

```shell
npx create-twilight-app
```
